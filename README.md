Sortable
========

HTML5 Sortable. No jQuery needed. Only .5KBs gzipped!

###
Based on [HTML5Rocks Demo](http://www.html5rocks.com/en/tutorials/dnd/basics/)

## Simple API

```
var s = Sortable({
	els: '.sortable' // Selector or array of elements
});


// destroy (not necessary)
s.destroy();
```
